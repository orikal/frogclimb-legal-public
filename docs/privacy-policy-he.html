1. תקציר

FrogClimb היא בעיקר משחק ללא שרת משחק ייעודי שלנו, ואין באפליקציה מערכת התחברות (חשבון משתמש) שאנו מפעילים. רוב נתוני המשחק (כגון שיאים, העדפות ומלאי פריטים במשחק) נשמרים מקומית במכשיר (למשל באמצעות אחסון מקומי של React Native Async Storage).

תכונות אופציונליות מסוימות מסתמכות על שירותי Apple או Google (למשל רכישות בתוך האפליקציה, ובמכשירי iOS נתמכים — Apple Game Center). עיבוד המידע שם כפוף למדיניות הפרטיות שלהם.

2. מידע המעובד במכשיר (אחסון מקומי)

האפליקציה עשויה לשמור במכשיר:

  • משחק והתקדמות: שיא גובה/ניקוד, מוני כוכבים או אסימונים, ומצב משחק קשור
  • חנות ושדרוגים: מלאי, שימושים חינמיים, ותאריכים לצורך זכאות לפרס יומי
  • הגדרות: סוג שליטה (כפתורים / הטיה / לחיצה בצדדים), שפה, שמע ורטט
  • קוסמטיקה: דמות ואביזרים נבחרים

מידע זה אינו נשלח אוטומטית לשרתים שלנו בהגדרת המוצר הנוכחית. מחיקת האפליקציה או ניקוי נתוני האפליקציה עשויים למחוק מידע זה, לפי מערכת ההפעלה.

3. חיישני מכשיר (שליטת הטיה)

אם תבחרו במצב הטיה, האפליקציה קוראת נתוני תנועה / תאוצה במכשיר בלבד כדי להזיז את הדמות. הנתונים משמשים לשליטה במשחק ולא לפרסום, ואינם נשלחים לשרתים שלנו (אין לנו שרת אנליטיקה פעיל למשחק כפי שפורסם כיום).

4. שמע

האפליקציה עשויה להשמיע אפקטים קוליים ומוזיקה. האפליקציה אינה מקליטה את המיקרופון.
